jChebi2gene
==========

:Author: Pierre-Yves Chibon <pierre-yves.chibon@wur.nl>, <pingou@pingoured.fr>


Java library to link a chebi compound or identifier to pathways
or tomato genes.


Get this project:
-----------------
Source:  https://github.com/PBR/jChebi2gene

-------------
